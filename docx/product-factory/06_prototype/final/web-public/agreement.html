为便于理解，本协议中使用的关键术语定义如下："服务"指本平台提供的、对接多家上游大模型供应商的统一 API 网关、密钥管理、计费、监控及配套功能；"上游模型"指由第三方模型厂商提供、经由本平台转发调用的大语言模型或多模态模型；"API 密钥"指本平台为您签发、用于身份鉴权与计量的凭证；"额度"指您预付费充值后形成的、用于抵扣调用费用的账户余额。

3服务内容与计费规则

本平台以统一接口形式聚合多家上游模型供应商的能力，您可通过单一密钥访问不同厂商的对话、补全、向量、图像及其他模型。具体可用模型清单、上下文长度及能力以控制台与模型广场页实时展示为准，可能随上游供应方调整而变更。

计费方式。本平台采用预付费模式，您需先充值形成账户额度，调用费用按实际消耗从额度中实时扣减。模型调用以 token 为基本计量单位，输入 token 与输出 token 通常分别计价；部分模型按请求次数、图像张数或时长计费，以该模型的计费说明为准。

倍率说明。每个模型设有相对于上游官方价格的倍率，最终单价 = 上游基准价 × 倍率。倍率用于覆盖网关运营、流量调度与汇率成本，并可能因促销或上游价格调整而变化。每次调用的实际扣费、对应模型与 token 数均记录在账单明细中，您可随时在控制台查询、核对与导出。

额度一经消耗不可逆转。除适用法律强制要求或我们书面承诺的情形外，已充值的账户额度原则上不予退还；如需退款，须经人工审核并扣除已消耗部分。我们保留在提前公告后调整定价、倍率及计费规则的权利，调整不溯及已发生的扣费。
