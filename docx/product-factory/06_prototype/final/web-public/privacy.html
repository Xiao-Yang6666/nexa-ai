隐私政策

本政策说明 Nexa·AI 在您使用平台服务过程中如何收集、使用、共享和保护您的个人信息，以及您对自身数据所享有的权利。我们承诺以最小必要原则处理数据。

1引言

Nexa·AI（以下简称"本平台""我们"）深知个人信息对您的重要性，并将依照所适用的数据保护法律法规，尽力保护您的隐私与数据安全。本隐私政策适用于您通过本平台网站、控制台及 API 接口使用服务时所涉及的个人信息处理活动。

本政策与《用户服务协议》共同构成您与我们之间关系的组成部分。当您注册账户或使用服务时，即表示您已阅读并理解本政策所述的信息处理规则。若您不同意本政策的任何内容，请停止使用相关服务。

2我们收集的信息

为向您提供稳定、安全的服务，我们仅在必要范围内收集以下类别的信息：

  • 账户信息：您注册时提供的邮箱、用户名、登录密码（以加盐哈希形式存储，我们无法还原明文），以及您可选填写的组织名称、联系方式。
  • API 调用元数据：每次请求的时间戳、所用模型、密钥标识、输入与输出 token 数量、请求来源 IP 与状态码。这些元数据用于计量、计费、限流与异常排查。
  • 计费记录：充值流水、扣费明细、发票抬头等交易相关信息。支付所需的银行卡或第三方支付账户信息由持牌支付服务商处理，我们不直接存储您的完整支付凭证。
  • 技术日志：浏览器类型、设备与操作系统信息、访问页面与时间等，用于保障安全、诊断故障与改进体验。
关于请求正文：默认情况下，我们不会持久化存储您 API 请求与响应的正文内容（如提示词、模型输出）。仅在排障所需且经您授权、或法律要求时，才会在受控、限时的前提下临时留存相关内容。

3信息使用方式

我们将所收集的信息用于以下合法、正当且与服务直接相关的目的：

  • 创建与管理您的账户、进行身份鉴权与密钥签发；
  • 计量调用用量、计算费用并生成账单与发票；
  • 提供限流、风控与安全防护，识别并阻止滥用、欺诈与攻击行为；
  • 向您发送服务通知、安全告警及您订阅的产品更新；
  • 在汇总、去标识化的前提下分析整体用量趋势，以优化网关性能与服务质量。

我们不会将您的请求正文内容用于训练我们或第三方的模型。任何超出上述目的的使用，我们将另行征得您的同意。

4数据共享与第三方

我们仅在以下必要情形下，与有限的第三方共享为完成服务所必需的信息，并要求其遵守不低于本政策的保护标准：

  • 上游模型厂商：当您调用某一模型时，您的请求内容会被转发至对应的上游供应商以生成响应。该转发受上游厂商各自的隐私与数据使用条款约束，我们建议您在调用敏感数据前知悉相关上游政策。
  • 支付服务商：充值与开票环节由持牌第三方支付机构处理，相关支付信息直接由其收集与保管。
  • 基础设施服务商：用于托管、存储与内容分发的云服务提供商，仅在为我们提供服务所必需的范围内接触数据。
  • 法律要求：当法律法规、司法或行政机关依法定程序要求时，我们可能依法披露相关信息。
我们不会出售您的个人信息，也不会为第三方的独立营销目的而共享您的信息。

5数据安全措施

我们采用与行业实践相符的技术与管理措施保护您的数据安全。传输层全程采用 TLS 加密，敏感字段（如登录凭证）以加盐哈希或加密形式存储，数据库与存储介质实施静态加密。

在访问控制方面，我们遵循最小权限原则，对内部系统实行基于角色的访问控制与操作审计；API 密钥经哈希存储且仅在创建时向您完整展示一次。我们建立了密钥轮换、入侵检测与定期安全评估机制，以持续降低风险。

尽管我们已尽合理努力，但任何安全措施都无法保证绝对安全。若发生可能危及您权益的数据安全事件，我们将依法及时启动应急响应，并通过适当方式通知受影响的用户与监管机构。

6数据保留期限

我们仅在实现本政策所述目的所必需的期间内保留您的个人信息。账户信息在您账户存续期间保留；账户注销后，除法律法规要求继续保留的情形外，我们将在合理期限内删除或匿名化处理相关信息。

调用元数据与技术日志通常保留有限期限以满足计费核对与安全审计需要；计费与交易记录则依据财税与会计法规所要求的最短期限保存。超过保留期限的数据将被安全删除或不可逆地去标识化。

7您的权利

在适用法律框架内，您对自己的个人信息享有以下权利，我们将提供便捷的行使途径：

  • 访问：查询我们持有的与您相关的账户信息及用量记录；
  • 更正：更新或修正不准确、不完整的个人信息；
  • 删除：在符合法定条件时请求删除您的个人信息或注销账户；
  • 导出：以结构化、通用格式获取您的账单明细与用量数据的副本。

多数权利可直接在控制台的"账户设置"中自助完成；如需进一步协助或行使删除、导出等权利，可通过本政策第 11 节的联系方式与我们联系。我们将在法定期限内核验身份后予以回应，原则上不收取费用。

8Cookie 与追踪

我们在网站与控制台中使用必要的 Cookie 及类似技术，以维持您的登录状态、保障会话安全并记住您的界面偏好。这些 Cookie 对于服务的正常运行不可或缺。

我们可能使用有限的、注重隐私的分析工具来统计页面访问与功能使用情况，以改进产品。这类分析以汇总、去标识化数据为主。您可通过浏览器设置管理或清除 Cookie，但禁用必要 Cookie 可能导致部分功能无法正常使用。我们不参与跨站点的广告追踪。

9儿童隐私

本平台面向开发者与企业用户，不面向未成年人提供服务。我们不会有意收集未满法定年龄（依所适用法律确定）未成年人的个人信息。

若您是监护人，发现未成年人在未经您同意的情况下向我们提供了个人信息，请通过第 11 节渠道联系我们，我们将在核实后及时删除相关信息并关闭对应账户。

10政策更新

我们可能因法律法规变化、业务调整或安全实践升级而不时修订本隐私政策。对于重大变更，我们将通过站内公告、邮件或控制台提示于生效前合理期限内通知您。

修订后的政策自公布的生效日期起适用，最新版本的生效日期标注于本页页头。建议您定期查阅本政策以了解我们对个人信息的最新处理方式。变更生效后您继续使用服务，即视为接受更新后的政策。

11联系我们

如您对本隐私政策或我们的个人信息处理活动有任何疑问、意见或投诉，或希望行使您的数据权利，可通过以下方式联系我们的隐私团队：

  • 隐私官邮箱：privacy@nexa.ai
  • 客户支持邮箱：support@nexa.ai
  • 控制台内"帮助与支持"工单系统（登录后可用）

我们将在收到您的有效请求后于合理且符合法律要求的期限内予以回复，并尽力妥善处理您的关切。感谢您信任 Nexa·AI。
